# enkripsiBase64
## Aplikasi Enkripsi Gambar Dengan Algoritma Base64 Update

Aplikasi ini dapa mengenkripsi gambar menjadi teks maupun sebaliknya medekripsi teks menjadi gambar dengan menggunakan algoritma base64
